# 创建 API 密钥

登录后进入控制台，左侧选择”API 密钥”。

![](/images/QuickStart/004.png)

### 进入 API 密钥

1. 在左侧菜单点击”API 密钥”。
2. 点击页面上方的”创建密钥”。

### 创建新密钥

在弹窗中填写密钥信息：

![](/images/QuickStart/005.png)

- 名称：用于区分不同用途，例如 `Claude Code`、`Codex`、`Gemini`。
- 分组：必须选择，分组决定这个密钥可以使用哪些模型。
- 密钥有效期：默认”永不过期”，也可以按需要设置有效期。
- 额度限制：设置此密钥可消费的最大金额，0 = 无限制。
- IP 限制：不熟悉时建议先保持默认，不要开启。

填写完成后，点击右下角”创建”完成创建。
