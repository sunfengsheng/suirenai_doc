概述 ​

这些服务特定条款（“服务特定条款”）包含适用于特定服务的补充条款，并构成您或您所代表的组织、公司或其他实体（“客户”）与 Suirenai 就引用或以其他方式引用这些服务特定条款的此类服务（例如商业服务条款）的协议的一部分（“协议”）。这些服务特定条款中未定义的任何大写术语将具有协议中规定的定义。如果条款与这些服务特定条款之间存在任何冲突，这些服务特定条款对相关服务具有控制权。

A. Suirenai 工作版（团队计划；企业计划） ​

客户必须告知其用户 (i) 他们正在访问管理的服务产品，使客户能够访问和控制客户及其用户提交给服务的数据，并且 (ii) 服务的使用受 Suirenai 的政策约束，包括隐私政策和使用政策。客户负责获得任何必要的用户同意并提供适用法律要求的所有用户通知。

B. Beta 服务 ​

Suirenai 可能会提供处于预发布、测试、预览或类似试用形式的服务（“Beta 服务”）。这意味着它们不适合生产使用，并按“原样”临时提供。Suirenai 不对客户使用或依赖 Beta 服务负责，也没有义务为其使用提供任何赔偿。Suirenai 对 Beta 服务的责任限于在前 12 个月内客户为服务支付的费用中的较小者 $1,000。

C. 微调服务 ​

Suirenai 可能会提供允许客户微调或进行其他形式的模型定制的服务（“微调服务”以及任何此类结果模型，即“微调模型”），包括使用客户提供的数据（“微调材料”）。微调服务使用微调材料来生成或验证微调模型。微调材料是客户内容。客户使用微调服务不构成违反协议中训练禁令；前提是进行的训练仅用于客户使用微调服务。

D. 云托管服务 ​

一般 ​

Suirenai 通过以下第三方管理和托管的技术环境提供某些服务：(1) Amazon Bedrock（“Bedrock”），由 Amazon Web Services, Inc. 及其关联公司（“AWS”）托管和管理；以及 (2) Vertex AI（“Vertex”），由 Google LLC 及其关联公司（“Google”）托管和管理。AWS 和 Google 是“云提供商”，其环境是这些服务特定条款的“云平台”。客户承认并同意 Suirenai 不对云平台或客户从云提供商处获得的任何其他服务负责，并且对云平台或其他云提供商服务的更改可能会严重影响客户对服务的使用。客户只能在遵守适用云提供商政策和其与适用云提供商的协议的情况下使用服务。

云平台上的内容 ​

Suirenai 提供的用于通过云平台访问服务的技术不会使 Suirenai 访问云提供商的任何客户实例，包括它们包含的客户内容。

费用 ​

客户必须按 Suirenai 当时的费率向适用的云提供商支付通过云平台使用服务相关的所有适用费用和税费，除非各方另有约定。费率更新在适用云提供商或 Suirenai 发布更新后 30 天或客户以其他方式收到通知后的较早日期生效。

E. 开发合作伙伴计划 ​

客户可以（自行决定）通过为服务启用许可数据选择加入设置（“开发合作伙伴模式”）来参与开发合作伙伴计划。如果客户启用开发合作伙伴模式，Suirenai 可以在与 Suirenai 的产品和服务相关的情况下使用客户提交给服务的数据（例如，客户内容），包括训练模型。

F. API 使用限制 ​

客户必须遵守以下 API 使用限制：

  • 速率限制： 客户必须遵守 Suirenai 规定的 API 调用速率限制
  • 配额： 客户的使用不得超过分配的配额
  • 缓存： 客户可以缓存响应，但必须遵守 Suirenai 的缓存政策
  • 监控： Suirenai 保留监控 API 使用情况的权利

G. 数据保留 ​

  • 客户数据： 客户数据将根据客户的保留设置进行保留
  • 备份： Suirenai 将维护系统备份以确保服务连续性
  • 删除： 终止后，客户数据将根据数据处理附录进行处理

H. 服务级别协议（SLA） ​

Suirenai 致力于提供高可用性服务：

  • 可用性目标： 99.9% 的月度正常运行时间
  • 维护： 计划维护将提前通知
  • 支持： 根据客户计划提供技术支持

I. 更新和修改 ​

Suirenai 可能会不时更新这些服务特定条款。重大更改将提前通知客户，并将在通知后 30 天生效。
